## 第1章

### 論理回路の作成
- すべてのデジタル機器は回路からできている
- この章では基本となる6つの論理回路を作成していく

#### ポイント：すべての論理回路はNANDゲートのみで実装できる
- ゲートと回路の実装方法はいくつも存在する。
  - NORゲートだけからスタートすることも、AND・OR・NOTゲートからスタートすることも可能
  - 実装方法によりスピード、消費電力、製造コストなどに違いがある。
